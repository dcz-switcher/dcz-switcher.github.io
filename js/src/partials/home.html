C'est quoi ce site ?

Ce site me permet de tester différentes choses comme le RWD, angularJS, Polymer etc.

Rien que l'hébergement est original puisqu'il utilise Github, ainsi, tout le code source est disponible.

Le site est codé sous différents éditeurs comme Brackets ou LightTable

Il ne s'agit pas d'un site conventionnel mais d'un Single Page Application qui tourne avec AngularJS
